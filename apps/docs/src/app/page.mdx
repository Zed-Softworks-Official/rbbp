# RBBP (Role-Based Ban Protection)
A Discord bot that automatically bans malicious bots attempting to exploit the onboarding process by self-assigning roles

## What is RBBP?
RBBP monitors your server for bots that assign themselves a specific roles during the onboarding process.
This is a common tatic used by malicious bots to gain elevated permissions.
When detected, the bot is automatically banned.

## Features
- **Role Protection**: Register roles that should never be self-assigned
- **Automatic Banning**: Instantly removes bots that trigger protection
- **Logging**: Track all protection events in a dedicated channel
